# Proyecto Pokemon Web

## Requisitos
- Java 11
- NetBeans
- Tomcat
- MongoDB Atlas

## Configuración MongoDB
1. Crear cluster en Mongo Atlas
2. Permitir IP: 0.0.0.0/0
3. Copiar URI en MongoConexion.java

## Ejecutar proyecto
1. Abrir en NetBeans
2. Clean and Build
3. Run
